# homebrew-idotenv
idotenv brew tap

Homebrew formula for installing idotenv.

## How To Use

Add repository:

```
brew tap h4ux/idotenv
```

Install [idotenv](https://github.com/h4ux/idotenv):

```
brew install idotenv
```

Upgrade the idotenv CLI to the latest version:

```
brew upgrade idotenv
```
